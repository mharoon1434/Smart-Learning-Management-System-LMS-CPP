<h1 align="center">🎓 Smart Learning Management System (LMS)</h1>

<p align="center">
  A Console-Based LMS built in C++ using Object-Oriented Programming
</p>

<hr/>

<h2>📖 Description</h2>
<p>
A console-based Learning Management System (LMS) built in C++ using Object-Oriented Programming, 
where instructors can create, update, and manage courses, and students can enroll, access study materials, 
complete assignments, and track their progress.
</p>

<hr/>

<h2>🚀 Features</h2>
<ul>
  <li>Instructor can create and manage courses</li>
  <li>Students can enroll in courses</li>
  <li>Access course materials and assignments</li>
  <li>Track student progress</li>
  <li>Console-based interactive system</li>
</ul>

<hr/>

<h2>🛠 Technologies Used</h2>
<ul>
  <li>C++</li>
  <li>Object-Oriented Programming (OOP)</li>
  <li>STL (Vectors)</li>
</ul>

<hr/>

<h2>⚙️ How to Run</h2>
<pre>
g++ main.cpp -o lms
./lms
</pre>
